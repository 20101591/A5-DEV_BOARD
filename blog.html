Question-1: What are the different ways to select an element in the DOM?

To select an element in the DOM, we can use getElement or querySelector method.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used to modify HTML content, innerText is used to modify visible texts and textContent can modify hidden text along with visible texts.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript approach that attaches a single event listener to a parent element rather than several child elements.

Question-4: What is event bubbling in the DOM?

The DOM (Document Object Model) includes the idea of Event Bubbling. It occurs when an element receives an event, which then bubbles up (or is communicated or propagated) through the DOM tree to its parent and ancestor elements until it reaches the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

The document.createElement() method can be used to create an element. For adding an element, appendChild() can be used whereas either the remove() or removeChild() methods can be used to eliminate an element.
